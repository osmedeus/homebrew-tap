# Homebrew Tap for Osmedeus

This is the official [Homebrew](https://brew.sh/) tap for [Osmedeus](https://github.com/j3ssie/osmedeus) - A Modern Orchestration Engine for Security.

## Installation

```bash
brew install osmedeus/tap/osmedeus
```

## Upgrade

```bash
brew update
brew upgrade osmedeus
```

## Uninstall

```bash
brew uninstall osmedeus/tap/osmedeus
```

## Usage

```bash
osmedeus --help
```
